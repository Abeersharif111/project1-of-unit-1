Project 1 of Unit 1

The project plan


User stories:

•	When I arrive the website of the game , I want to see a landing page with clear name of the game and  so I know I'm in the right place.
•	I want to see the 3 lollypops  that  centered horizontally and a kid is standing with a basket .
•	I want to have a button to start the game , so the game timer will start , and the colors of the lollypops will start changing it’s colors randomly.
•	the player must click only the red lollypops so the score will increment by 1and I want a message tell me that I am doing correctly , if he click 3 red lollypops or more during the 15 seconds then he will win, if less than 3 then he lose. 
•	I want to be presented with a clear message indicating the winner of the game.

Advanced editions to the game if possible :
•	at the beginning of the game the basket will be little far from the kid , as much as he click red lollypops the basket will be moved near a step near to the kid and the score will incremented by 1 .
•	If he clicked a blue lollypop, the basket will move far a step of the kid and the score will decreased by 1 and I want a message indicating that I have a fault.
•	If he clicked the black lollypop the score becomes to 0 and a message of a big fault  must indicated.


The pseudocode :

1) Define any variables used to track the game:
    The timer
    The match score - win/lose
    An encourage message - displays while clicking red lollypops
    A result message - display who won that round

 2) Define the required constants:  
  The 3 lollypops.
  The basket.

 3) Handle a player clicking the start button 

 4) Handle a player clicking the lollypops.

 5) Store cached element references.

 6) Upon loading, the game timer should be initialized, and a function should 
   be called to render this game .

 7) Handle the game over logic. 

 8) Create reset functionality.
