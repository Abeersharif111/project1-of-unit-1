## Your game's name:
Lollypop Hunter Game . 
As you a player you must help this kid to collect 3 or more lollypops to win the game , if 
you collect less than 3 lollypops then you will lose. 
If you want to play again you must click the Play Again button to restart the game. 
I choose this game because I can use the basic fundamentals and then I can go more 
advanced .
